1.	composer install   需要安装composer之后才能执行
2.	cp .env.example .env          复制.evn的例子,[数据库文件]
3.	php artisan key:generate   生成唯一密钥,可以多次生成.否则报错
4.	Create database on MySQL   创建空白数据库
5.	Set database credentials on .env file    修改.evn文件
6.	php artisan migrate   还原数据库命令
7.	php artisan storage:link    创建一个软连接 public/storage（软连接，指向目录 `storage/app/public`）
8.	php artisan serve  启动了一个什么脚本,创建一个新网站18896647173
